문자열 처리

문자열 처리 팁

  1. 문자열 처리는 다른 자료형에 매핑하는 것 보다 그냥 문자열 그대로 사용하는 것이 제일 빠르다
  2. 파이썬에서 문자열 슬라이싱은 강력하면서도 빠른 연산을 제공한다. 최대한 슬라이싱으로 처리하는 것이 좋다
  3. 함수 먼저 구현하려고 하지 말고 이 기능을 제공하는 내장함수가 있는지부터 생각하자
    • 문자열/리스트의 순서를 통쨰로 뒤집는 것은 객체.reverse()함수나 [::-1]슬라이싱을 이용하자
    • 어떤 문자를 기준으로 리스트로 쪼개는 것은 문자열.split(), 합치는 것은 문자.join(리스트)를 이용하자
