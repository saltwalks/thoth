입출력함수

화면 출력 : print()

  1. 문자열을 띄어쓰기로 연결하면 + 연산이 된다
  2. 문자열을 콤마로 연결하면 띄어쓰기가 추가되어 + 연산이 된다
  3. 기본적으로 줄바꿈이 추가로 수행된다
    • print()함수의 생략 가능한 매개변수로 end가 있는데 이것의 디폴트값은 end='\n'이기 떄문
    • 이것을 딴것으로 바꿔주려면 end="하고싶은거"로 매개변수 입력값을 바꿔주면 된다

키보드 입력 : input()

  1. input()함수의 리턴값은 키보드로부터 입력받은 값이고, 입력버퍼 앞에 띄울 안내 문구를 입력값으로 넣어줄 수도 있다
  2. sys라이브러리를 이용하면 파이썬 파일 실행시에 인자들을 입력받아올 수 있게 할 수 있다
    • 터미널로 파이썬 파일 실행시에 "python 파일이름.py"로 실행하게되는데
    • 이때 python 파일이름.py 인자1 인자2 ... 이렇게 받아올 수 있다
    • sys 모듈을 import하면 "python"명령어 다음에 등장하는 요소들을 sys.argv라는 리스트에 차례로 담는다
    • 즉, sys.argv = ["파일이름.py", "인자1", "인자2", ... ] 이렇게됨
    • 이것을 인덱스로 접근하던 슬라이싱을 하던 해서 사용하면 되더라

파일 열고 닫기

  1. 파일 열기 : open(파일경로, 읽기방식)
    • 파일경로는 문자열 형태("")로 입력받는다
    • 읽기방식 : 'r' = 파일 읽기 전용, 'w' = 파일 만들기, 기존파일 처음부터 다시 입력하기, 'a' = 기존파일 수정하기
  2. 파일 닫기 : close()
    • 메모리 최적화와 오류 방지를 위해 안쓰면 꼭 닫아놓자
  3. 열기와 닫기를 한번에 : with open("파일경로", "읽기방식") as 파일객체변수이름 :
    • 인덴트를 이용해 with문이 끝나면 자동으로 파일을 닫아주는 역할

파일 입/출력

  1. 파일 입력 : 파일객체.write(입력값)
  2. 파일 출력1 : 파일객체.readline()
    • 파일의 한 줄을 읽어온다. 각 줄을 인덱스로 접근하는게 아니라 iterate기능으로 불러올때마다 읽어온줄의 다음줄을 읽어온다
    • 따라서 첫 한줄만 읽어올게 아니면 반복문과의 조합이 필수적이다
  3. 파일 출력2 : 파일객체.readlines()
    • 파일 내의 모든 내용을 줄별로 리스트로 만들어 반환한다
    • 따라서 인덱스로 접근할 수 있음
  4. 파일 출력3 : 파일객체.read()
    • 파일 내의 모든 내용을 하나의 문자열로 받아온다
