문자열

표현

  1. 작은따옴표도 문자열에 사용할 수 있음
  2. 큰/작은 따옴표 세번을 사용해서 문자열을 표현할 수 있음 = ‘’’abc’’’, “””abc”””
    • 이건 문자열에 큰/작은 따옴표가 들어갈때 오해의 여지를 없애고자 사용
    • 또한 세번 사용하면 \n 없어도됨 = 그냥 엔터 누르면 자동으로 \n 이 들어가는 효과

인덱스

  1. 문자열/리스트/튜플 등의 인덱스 참조에 음수 사용 가능 = 뒤에서부터 온다
  2. 슬라이싱 = [시작인덱스:끝인덱스]로 여러 항목을 가져올 수 있음
    • 인덱스 생략하면 자동으로 맨앞, 맨뒤 인덱스로 설정된다
    • 끝인덱스는 포함 안된다

포매팅

  1. 퍼센트 포매팅의 기본구조 : ("%수식" % 넣을값)
  2. 퍼센트 포매팅 숫자수식 : %숫자s 이렇게 하면 포매팅 되는 부분 총 길이가 ‘숫자’ 가 됨
    • 공백은 스페이스로 채워진다
    • 숫자가 양수일 경우 공백이 아닌 문자는 오른쪽 정렬됨, 음수는 왼쪽 정렬
    • 소숫점을 이용해 소숫점 뒤 몇자리까지 표현하는지 가능, 다만 이 경우 f를 붙여야된다
  3. format()함수 = 문자열 안에 {}을 이용해서 인덱스 형식으로 포매팅 가능
    • “{0} {1} {2}”.format(1, 2, 3) 일케하면 인덱스 0, 1, 2에 1, 2, 3이 포매팅된다
    • 인덱스 말고 이름으로 지정도 가능 = “{aa}, {bb}, {cc}”.format(aa=1, bb=2, bb=3) 이런식으로
    • {인덱스 : 공백처리 정렬방식 포매팅길이} 이렇게 사용 가능
    • 공백처리는 안쓰면 스페이스, 공백을 *로 채우고 싶으면 * 이런식으로 넣어주면됨
    • 정렬방식은 <은 왼쪽정렬 ^은 가운데정렬 >은 오른쪽 정렬이다
    • 포매팅길이, 소숫점은 퍼센트 포매팅에서와 동일(소숫점의 경우 f 꼭 붙여라)
  4. f. 포매팅 = 문자열 안에 {변수이름}을 직접 포매팅하고자 할때 사용
    • >{변수명 : 공백처리 정렬방식 포매팅길이} 마찬가지로 사용 가능
