조건/반복문

조건문

  1. 파이썬에서의 else if 는 elif이다
  2. 파이썬에서의 조건부 표현식은 (변수 = 값 if 조건 else 값) 이렇게 쓰면 된다
  3. pass 는 해당 조건에 암것도 안하고 넘어가는 경우이다
  4. in, not in 연산자 : a in b 는 a 가 b라는 리스트, 튜플 등의 요소이면 참이고 아니면 거짓. Not in은 그 반대

반복문

  1. for문 기본구조 : for 변수 in 반복가능객체
    • for문에서의 in은 진위여부 연산이 아니라 iterate연산이다
    • 즉, 뒤의 객체의 인덱스 순번대로 하나씩 앞의 변수에 담기면서 for문이 굴러가게 된다
  2. for문에서의 변수는 꼭 하나가 아니어도 된다. 튜플들을 요소로 갖고있는 리스트를 탐색하려할때 (a, b)이런식으로 튜플 변수군을 변수로서 넣어줘도 된다
  3. for문 기본구조 2 : for 변수 in range(범위)
    • range(시작, 끝) 함수 : range는 시작~끝-1까지의 연속된 정수를 리스트로 만들어주는 함수이다
    • 물론 리스트 형태로 출력되지 않고 range객체로 나오지만 리스트 생성자를 통해 변환도 가능
    • 이것을 for문과 연결해 for 변수 in range(시작, 끝) 일케 하면 변수가 저 범위의 모든 정수를 다 훑게 할 수 있다.
  4. 리스트 내포 : 파이썬에서는 리스트 내에 for문을 포함시킬 수 있다
    • 기본구조 : [수식 for 수식 내의 변수 in 리스트(같은 반복가능객체) if 조건식도 포함 가능]
