라이브러리

자주쓰는 라이브러리 정리

  1. picle : 파일에 문자열형태로 저장하는것이 아닌 객체형태 그대로 저장하는 것
  2. os : 시스템적인 일들을 처리해주는 것
    • os.chdir() : 현재 디렉터리 위치 변경
    • os.getcwd() : 현재 디렉토리 출력
    • os.system() : 터미널 명령어 실행
    • os.popen() : 터미널 명령어를 실행해서 결과값을 읽기모드 파일객체로 반환
  3. shutil : 파일 복사
  4. lob : 특정 디렉토리에 있는 파일 목록을 알고싶을때
  5. tempfile : 임시 파일 만들기
  6. webbrowser : 기본 웹브라우져 실행
  7. pprint : 가독성있게 출력(local()와 같이 주로 활용
