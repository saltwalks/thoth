모듈

모듈이란

  1. 파이썬에서는 다른 파이썬 파일을 내 파일로 가져와서 해당 파일에 명시돼있는 변수, 함수, 클래스 등을 사용할 수 있다. 이때의 파이썬 파일이 모듈이다

모듈 가져오기

  1. import 경로
    • import를 사용하고자 할때는 경로의 도착지가 모듈이 되어야 한다
  2. from 경로 import 가져올놈
    • from-import같은 경우에는 경로의 도착지가 모듈이 아닌 디렉토리가 되어도 된다
    • 도착지가 모듈일 경우에는 import뒤에 모듈 내의 특정 변수, 함수, 클래스 등을 적어서 일부분을 가져오거나 *를 적어서 모듈 내의 모든 내용을 가져올 수 있다
    • 도착지가 디렉토리일 경우에는 import뒤에 모듈명을 적어서 특정 모듈을 가져올 수 있다
    • import뒤에 *를 적어서 디렉토리의 여러 모듈을 가져올 수도 있지만 이때는 __init__.py파일의 __all__변수를 설정해줘야 한다
    • __init__.py 파일과 __all__변수는 다음장의 패키지에서 배운다

경로에 대하여

  1. 모듈이 현재 디렉토리에 있거나 현재 디렉토리 하부의 다른 디렉토리 상에 있으면 따로 작업을 쳐주지 않아도 된다
    • 현재 디렉토리에 있으면 경로=모듈명이 될 것이고
    • 하부 디렉토리에 있으면 경로=하부디렉토리.모듈명이 될 것이다(하부 디렉토리가 여러 단계여도 .을 이용해 드가면된다)
  2. 하지만 현재 디렉토리가 아닌 다른 디렉토리에 있으면 path추가 작업을 해줘야 된다
    • 무슨말인고 하니 현재디렉토리/현재디렉토리의 하부디렉토리는 이미 sys.path에 들어가있어서 따로 추가를 안해줘도 됐지만
    • 현재디렉토리가 아닌 외부디렉토리는 sys.path에 추가가 안돼있어서 파이썬이 검색을 못한다
    • path는 파이썬이 모듈을 찾으러 방문해볼 장소라고 생각하면 된다
    • path추가방법은 바로 아래에 있음
    • path를 추가하고 난 뒤에는 마찬가지로 path에 모듈이 바로 있으면 그냥 모듈명을 쓰면 되고, 하부 디렉토리에 있으면 .로 모듈이 있는 디렉토리로 들어가면서 경로를 명시해주면 된다

path 추가하기

  1. 파이썬 파일 내에서 추가하기 : import sys 하고 sys.path.append("외부디렉토리 경로")
    • sys.path는 모든 path들이 리스트로 저장되어있다. 여기에 append()를 이용해 내가 필요한 모듈이 위치한 외부 디렉토리의 path를 추가해 주는 거다
  2. 터미널에서 추가하기 : set PYTHONPATH=외부디렉토리 경로
    • 터미널 명령어 set을 이용해 PYTHONPATH=경로 이렇게 지정해주면 알아서 sys.path에 해당 경로가 추가된다
  3. 여기서 주의할점 : 경로를 홈디렉토리를 이용해 ~/어쩌고 이렇게 적으면 에러난다 - 왠지모름

가져온 모듈 이용하기

  1. 이것만 기억해라 : (import뒤에 적어준 내용).(원하는거) 이렇게 적어주면 된다
    • 예를들어 디렉토리 A에 모듈B가 있고 함수 C()를 쓰려고 한다면
    • import A.B => A.B.C()
    • from A.B import C => C()
    • from A import B => B.C()
    • 이런식으로 해주면 된다 이말이야
  2. as를 이용하면 닉네임을 설정해줘서 호출할 수도 있다
  3. 위의 예시를 빌려서 import A.B as nickname 이렇게 해주면 nickname.C()이렇게 호출할 수도 있다

__name__변수

  1. 모듈 내에 모듈로써 동작하길 원하지 않는 내용이 있다면 if __name__ == "__main__" : 조건문을 이용해 주면 된다
    • import를 하는 즉시 그냥 복붙이 아니라 안의 내용이 실행된다 - 파이썬이 바로바로 한줄씩 인터프리트 하는 언어여서 그런듯
    • 즉, 모듈 안에 예를들어 print()문이 있으면 import하는 즉시 print()문이 실행될것이다
    • 또한 모듈 내에 직접실행시에는 동작해도 다른 파일에 모듈로써 포함될때에는 동작을 원하지 않는 내용들도 있을 수도 있다
    • __name__함수는 파이썬 내장 변수로, 해당 파일이 직접실행되었는지 모듈로 실행이 되었는지가 저장된다
    • 직접실행된 경우에는 __name__변수에 "__main__"이 들어가고 모듈로 실행되면 "모듈명"이 들어간다
    • 따라서 저 조건문으로 실행요건에 따른 동작여부를 결정해 줄 수 있다
