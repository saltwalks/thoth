내장함수

자주 쓰는 내장함수 정리

  1. all() : and 연산
  2. any() : or 연산
  3. divmod() : 두 숫자 나눗셈의 몫, 나머지를 튜플로 묶어서 반환
  4. enumerate() : 리스트, 튜플 등의 인덱스참조 자료형을 받아 각 요소와 그것의 인덱스를 튜플로 묶어 튜플리스트로 반환한다
  5. eval() : 파이썬 명령어를 실행시켜 예) eval("1+3") = 4
  6. filter() : 리스트같은 반복가능객체의 각각 요소들을 하나씩 함수에 넣고 그 결과값중 참인것만 반가객으로 묶어서 반환함
  7. instance() : 입력한 객체가 입력한 클래스의 인스턴스인지 확인해줌
  8. map() : filter랑 비슷하지만 filter는 참인것만 묶어서 반환하지만 map은 그냥 결과값을 반환함
  9. pow() : x의y승
  10. round() : 반올림
  11. sum() : 리스트나 튜플의 요소들 합
  12. zip() : 요소의 갯수가 같은 리스트/튜플들을 받아 같은 인덱스인놈들끼리 튜플로 묶어 튜플리스트로 반환함
  13. local() : 현단계에서 모든 로컬 변수의 이름과 그에따른 값을 딕셔너리 형태로 묶어서 반환. 디버깅할떄 좋다
